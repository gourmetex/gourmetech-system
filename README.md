## 📌 Sobre o Projeto

O Gourmetech é um sistema ERP completo focado no setor de comércio e alimentação. Desenvolvido sob a marca da KSI, a plataforma centraliza a operação do negócio, oferecendo desde o controle de caixa no PDV até a gestão inteligente de estoque e finanças.

O modelo do produto é estruturado para facilitar a adesão do lojista: sem mensalidades fixas, operando através de uma pequena taxa percentual sobre as vendas processadas.

## ✨ Funcionalidades Principais

- **PDV (Ponto de Venda) Ágil:** Interface otimizada para fechamento rápido de vendas e processamento de pagamentos.
- **Gestão de Mesas e Comandas:** Controle do salão em tempo real, permitindo lançamentos simultâneos e divisão de contas.
- **Controle de Estoque Inteligente:** Baixa automática de insumos a cada venda e monitoramento de reposição.
- **Gestão Financeira:** Fluxo de caixa detalhado, relatórios avançados de faturamento e integração nativa com gateways (Pagar.me / Stripe).
- **Ecossistema de Delivery:** Integração nativa com plataformas de entrega, como o Compra Rápida.
